=head1 NAME

Net::FastCGI::Record::EndRequest - Object representation of a C<FCGI_EndRequestRecord>

=head1 SYNOPSIS

    # Constructors
    $record = Net::FastCGI::Record::EndRequest->new($request_id, $application_status, $protocol_status);
    $record = Net::FastCGI::Record::EndRequest->parse($request_id, $octets);
    
    # Methods
    $application_status = $record->get_application_status;
    $protocol_status    = $record->get_protocol_status;

=head1 DESCRIPTION

=head1 INHERITANCE

C<Net::FastCGI::Record::EndRequest> extends L<Net::FastCGI::Record>.

=head1 CONSTRUCTORS

=head2 new

Constructs an instance of C<Net::FastCGI::Record::EndRequest>.

I<Usage>

    $record = Net::FastCGI::Record::EndRequest->new($request_id, $application_status, $protocol_status);

I<Arguments>

=over 4

=item C<$request_id>

An unsigned 16-bit integer.

=item C<$application_status>

An unsigned 16-bit integer.

=item C<$protocol_status>

An unsigned 8-bit integer.

=back

I<Returns>

=over 4

=item C<$record>

An instance of C<Net::FastCGI::Record::EndRequest>.

=back

=head2 parse

Parses C<FCGI_EndRequestBody>.

I<Usage>

    $record = Net::FastCGI::Record::EndRequest->parse($request_id, $octets);

I<Arguments>

=over 4

=item C<$request_id>

An unsigned 16-bit integer.

=item C<$octets>

A string of octets containing the body, must be greater than or equal to 8 octets in length.

=back

I<Returns>

=over 4

=item C<$record>

An instance of C<Net::FastCGI::Record::EndRequest>.

=back

=head1 METHODS

=head2 get_application_status

I<Usage>

    $application_status = $record->get_application_status;

I<Returns>

=over 4

=item C<$application_status>

An unsigned 16-bit integer.

=back

=head2 get_protocol_status

I<Usage>

    $protocol_status = $record->get_protocol_status;

I<Returns>

=over 4

=item C<$protocol_status>

An unsigned 8-bit integer.

=back

None.
