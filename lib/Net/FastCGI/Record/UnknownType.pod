=head1 NAME

Net::FastCGI::Record::UnknownType - Object representation of a C<FCGI_UnknownTypeRecord>

=head1 SYNOPSIS

    # Constructors
    $record = Net::FastCGI::Record::UnknownType->new($unknown_type);
    $record = Net::FastCGI::Record::UnknownType->parse($octets);
    
    # Methods
    $unknown_type = $record->get_unknown_type;

=head1 DESCRIPTION

=head1 INHERITANCE

C<Net::FastCGI::Record::UnknownType> extends L<Net::FastCGI::Record>.

=head1 CONSTRUCTORS

=head2 new

Constructs an instance of C<Net::FastCGI::Record::UnknownType>.

I<Usage>

    $record = Net::FastCGI::Record::UnknownType->new($unknown_type);

I<Arguments>

=over 4

=item C<$unknown_type>

An unsigned 8-bit integer.

=back

I<Returns>

=over 4

=item C<$record>

An instance of C<Net::FastCGI::Record::UnknownType>.

=back

=head2 parse

Parses C<FCGI_UnknownTypeBody>.

I<Usage>

    $record = Net::FastCGI::Record::UnknownType->parse($octets);

I<Arguments>

=over 4

=item C<$octets>

A string of octets containing the body, must be greater than or equal to 8 octets in length.

=back

I<Returns>

=over 4

=item C<$record>

An instance of C<Net::FastCGI::Record::UnknownType>.

=back

=head1 METHODS

=head2 get_unknown_type

I<Usage>

    $unknown_type = $record->get_unknown_type;

I<Returns>

=over 4

=item C<$unknown_type>

An unsigned 8-bit integer.

=back

=head1 EXPORTS

None.
